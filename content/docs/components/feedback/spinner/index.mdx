# Spinner

Spinners provide a visual cue that an action is processing awaiting a course
of change or a result.

<ComponentLinks
  github={{ package: "spinner" }}
  npm={{ package: "@chakra-ui/spinner" }}
  theme={{ componentName: "spinner" }}
/>

## Import

```js
import { Spinner } from '@chakra-ui/react';
```

## Usage

```jsx live
<Spinner />
```

### Spinner with Color

```jsx live
<Spinner color='red.500' />
```

### Spinner with different size

```jsx live
<Stack direction='row' spacing={4}>
  <Spinner size='xs' />
  <Spinner size='sm' />
  <Spinner size='md' />
  <Spinner size='lg' />
  <Spinner size='xl' />
</Stack>
```

### Spinner with empty area color

```jsx live
<Spinner
  thickness='4px'
  speed='0.65s'
  emptyColor='gray.200'
  color='blue.500'
  size='xl'
/>
```
